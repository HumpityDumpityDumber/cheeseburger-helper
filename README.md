<h3> declaritive and innovative text editing in one binary! </h3>

# Installation

run
```nushell
go build
```
to build the program


# Usage
Run it with 
```nushell
./gary-text-editor
```

press Ctrl+S to save your content

To open prior saved content run
```nushe;;
./gary-text-editor -file gary-output
```
